# THRESHOLDING

## Aim

To segment the image using global thresholding, adaptive thresholding, and Otsu's thresholding using Python and OpenCV.

---

## Learning Objective

- Understand different image thresholding techniques  
- Learn how image segmentation is performed using OpenCV  
- Compare global, adaptive, and Otsu’s thresholding methods  

---

## Software Required

- Anaconda – Python 3.7  
- Jupyter Notebook / VS Code  
- OpenCV (cv2)  
- NumPy  
- Matplotlib

## Algorithm

### Step 1:
Import the necessary libraries such as OpenCV, NumPy, and Matplotlib.

### Step 2:
Read the input image and convert it into grayscale format.

### Step 3:
Apply global thresholding to segment the image using a fixed threshold value.

### Step 4:
Apply adaptive thresholding to segment the image based on local neighborhood intensity values.

### Step 5:
Apply Otsu’s thresholding method to automatically determine the optimal threshold value and display the results.

---

## Output

### Original Image
- Displays the original grayscale input image.

### Global Thresholding
- Segments the image using a fixed threshold value.

### Adaptive Thresholding
- Segments the image based on local pixel intensity variations.

### Optimum Global Thresholding using Otsu's Method
- Automatically determines the best threshold value for image segmentation.


## Result

Thus, the images are successfully segmented using global thresholding, adaptive thresholding, and optimum global thresholding using Python and OpenCV.
